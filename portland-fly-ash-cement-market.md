<p><p>Portland Fly Ash Cement is available in two types: Cement with Class F Fly Ash and Cement with Class C Fly Ash. Cement with Class F Fly Ash is made using fly ash with high calcium oxide content, while Cement with Class C Fly Ash is made using fly ash with lower calcium oxide content. Both types of cement contain fly ash as a supplementary cementitious material, which helps improve the durability and strength of the concrete. These types of cement are commonly used in construction projects to reduce the environmental impact of concrete production.</p></p>
